Generic Parser
==============

The `GenericParser <#crawler.web_crawler.parsers.GeminiParser.GeminiParser>`_ class extracts product price and currency information from the HTML page using Gemini Ai:

* **Currency Extraction:** The `get_currency() <#crawler.web_crawler.parsers.GeminiParser.GeminiParser.get_currency>`_ method extracts the currency from the html page using the Gemini Ai.
* **Price Extraction:** The `get_price() <#crawler.web_crawler.parsers.GeminiParser.GeminiParser.get_price>`_ method extracts the price from the html page using the Gemini Ai.
